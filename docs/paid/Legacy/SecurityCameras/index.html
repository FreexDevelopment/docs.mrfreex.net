Frequent Issues​

Config Errors​

Most common error
SCRIPT ERROR: ?:-1: attempt to index a nil value (field '?')

This error can be the result of different issues, the most common one is that there is a syntax error in the config file. Search for missing comas and double text indexes. For more information visit https://www.lua.org/pil/2.5.html (complete but hard to understand) or https://create.roblox.com/docs/scripting/luau/tables (very easy to understand).
