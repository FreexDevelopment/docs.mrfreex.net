## How-To
Security Cameras can be placed by whoever possesses the Camera Item.<br/>
When used, the item shows an UI with arrow keys and a slider to move and rotate the camera.

<img src={PlacingMenu} />

After you click the <FontAwesomeIcon icon={faCheck} /> icon, a menu with the placement options will show.

<img src={FinishPlacing} />
<br />
<br />

:::caution

Only the *zones* you are authorized for will show up.
See "Configuring the zones" for more info.

:::
